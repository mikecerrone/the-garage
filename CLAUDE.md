# CLAUDE.md

## gstack

Use the `/browse` skill from gstack for all web browsing. Never use `mcp__claude-in-chrome__*` tools.

Available gstack skills:
- `/plan-ceo-review` — Review plan from a CEO/product perspective
- `/plan-eng-review` — Review plan from an engineering perspective
- `/review` — Code review
- `/ship` — Ship changes (commit, push, PR)
- `/browse` — Headless browser for QA, testing, and site verification
- `/retro` — Retrospective on completed work
